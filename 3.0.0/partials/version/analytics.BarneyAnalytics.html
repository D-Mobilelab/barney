BarneyAnalytics
service in module analytics

Description

Angular service of Analytics module

Import & Usage

To use Analytics service, you have to add BarneyAnalytics dependency to your component (directive, controller...). 

angular.module('mock').controller('HomePageController', [
    'BarneyAnalytics',
    function(BarneyAnalytics){
        // we can use "BarneyAnalytics" object here
    }
]);

Methods

  • init()

    Refer to init method of Analytics module, replacing barney.analytics to BarneyAnalytics

  • setDimension()

    Refer to setDimension method of Analytics module, replacing barney.analytics to BarneyAnalytics

  • setId()

    Refer to setId method of Analytics module, replacing barney.analytics to BarneyAnalytics

  • trackEvent()

    Refer to trackEvent method of Analytics module, replacing barney.analytics to BarneyAnalytics

  • trackPage()

    Refer to trackPage method of Analytics module, replacing barney.analytics to BarneyAnalytics
