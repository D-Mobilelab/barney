Dict

Dict is the module of Barney to show dictionary keys. You can use it:

  • as a provider
  • as a filter
  • as a directive

Usage

To use Dict module you have to include the Dict javascript file in your index.html:


  <!-- Dict module - Required -->
  <script type="text/javascript" src="./bower_components/barney/dist/dict.min.js">

Init Dict

Before using Dict module, it must be initialized.To initialize Dict module use Dict' s method init.

Methods:

  • get
  • init
  • list
