Welcome to Barney!

To use Barney library first of all you have to include main.min.js in your index.html file. You can find this file in the dist folder of Barney.

index.html:

 //Include Barney script
 <script type="text/javascript" src="../path/to/barney/dist/main.min.js">

Once you have included Barney in to your index.html file remember that you have to import Barney module as a dependency in your app.js file.

app.js: 

angular
.module('yourAngularApp', [
  ...,
  'barney',
  ...
])
