Analytics

Analytics is the module of Barney for Google Analytics. You can use it to:

  • set user id
  • set a session/user custom dimension
  • track a page
  • track an event

Usage

To use Analytics module you have to include the Analytics javascript file in your index.html:


  <!-- Analytics module -->
  <script type="text/javascript" src="./bower_components/barney/dist/analytics.min.js">

Init Analytics

Before using Analytics module, it must be initialized. To initialize Analytics you can use the Analytics method init.

Methods:

  • init
  • setDimension
  • setId
  • trackEvent
  • trackPage
