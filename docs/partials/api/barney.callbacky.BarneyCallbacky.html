BarneyCallbacky
service in module barney.callbacky

Description

Use Callbacky service

To use Callbacky service, you have to add BarneyCallbacky dependency to your component (i.e: directive, controller...). In this example, I have added dependency of BarneyCallbacky to a controller: 

angular.module('mock').controller('HomePageController', [
    'BarneyCallbacky', '$scope',
    function(Callbacky, $scope){
        // we can use "Callbacky" object here
    }
]);
Note that I included BarneyCallbacky as dependency but I have renamed it as Callbacky to use it more easily in controller code.

Methods

  • bind(key, method)

    Bind one function to a string.

    By this way, on the trigger event (with the relative string), the bound function can be called everywhere in your code.

    Parameters
    ParamTypeDetails
    keystring

    Unique key to identify the function

    methodmethod

    Function to call when triggered

    Example

    Bind

    Here is an example of the bind method. 

    
 Callbacky.bind('hello', function(args){ 
      console.log(args)) 
      };

    Remember: You have to do the bind of a function only once!

  • clean(key)

    Remove the bound function from the given string

    Parameters
    ParamTypeDetails
    keystring

    Unique key to identify the function

    Example

    Clean

    Here is an example of the clean method (the bind is in the example of bind docs). 

    
 Callbacky.clean('hello');

  • init(options)

    This method is used to initialize the Callbacky module and it requires an object that contains two values:

    • verbose: if true, log all the instruction done by the module, else logs nothing (default: false);
    • logger: Object that logs.
    Parameters
    ParamTypeDetails
    optionsObject

    (see attributes below)

    PropertyTypeDetails
    verboseboolean
    • true: the Callbacky module logs all the instruction done;
    • false: the Callbacky module logs nothing.

    (default: false)

    loggerObject

    Object that logs (i.e: window.console, BarneyLogger, ...)

    (default: null)

    Example

    Config Init

    Here is an example of the init method. 

    
  Callbacky.init({
      verbose:true, //it logs all the work
      logger: window.console //set as a logger the window.console logger
   });

  • trigger(key, arg)

    Call the function bound with the given string.

    On the trigger event, the corresponding function is called.

    Parameters
    ParamTypeDetails
    keystring

    Unique key to identify the function

    arg*

    Arguments of the function;

    Example

    Trigger

    Here is an example of the trigger method (the bind is in the example of bind docs). 

    
 Callbacky.trigger('hello', 'world');
    (After the bind) It logs 'world'.

    Remember: before call trigger somewhere in your code you have to do the relative bind method!
