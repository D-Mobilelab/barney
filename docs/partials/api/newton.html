Newton

Newton is the module of Barney that allows you to track events and pages with Newton library

Components

  • BarneyNewton

Usage

To use Newton module you have to include the Newton javascript file in your index.html:


  <!-- Newton module -->
  <script type="text/javascript" src="./bower_components/barney/dist/newton.min.js">

Init Newton

Before using Newton module, it must be initialized. To initialize Newton you can use the Newton method init.
