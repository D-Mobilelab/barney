callbacky

Description

You can use callbacky to bind and trigger a function from different points in your code (i.e: controllers, services, ...).

Import & Usage

Vanilla JS

Import this single file to use Callbacky module, in Vanilla JS: 

<script type="text/javascript" src="./bower_components/barney/base/callbacky.js">
Now you can use global object barney.Callbacky and associated methods, described below.
You can start by initializing it with init method.

Angular

Import this single file to use Callbacky module, in Angular framework: 

<script type="text/javascript" src="./bower_components/barney/angular/callbacky.js">
Now you can import barney module for your Angular app. 
angular.module('yourApp', [ 'barney' ]);
Callbacky module for Angular contains only BarneyCallbacky service.

Methods

  • bind(key, method)

    Bind one function to a string.

    By this way, on the trigger event (with the relative string), the bound function can be called everywhere in your code.

    Parameters
    ParamTypeDetails
    keystring

    Unique key to identify the function

    methodmethod

    Function to call when triggered

    Example

    Trigger

    Here is an example of the bind/trigger method 

    // FIRST FILE
barney.Callbacky.bind('hello', function(args){ 
    console.log(args)) 
};

// SECOND FILE
barney.Callbacky.trigger('hello', 'world');
// console.log('world');

  • clean(key)

    Remove the bound function from the given string

    Parameters
    ParamTypeDetails
    keystring

    Unique key to identify the function

    Example

    Clean

    Here is an example of the clean method 

    barney.Callbacky.clean('hello');

  • init(options)

    Init callbacky module.

    Parameters
    ParamTypeDetails
    optionsObject

    (see attributes below)

    PropertyTypeDetails
    verboseboolean

    enable/disable verbose mode.

    (default: false)

    loggerObject

    Object that logs (i.e.: window.console, BarneyLogger, ...)

    (default: null)

    Example

    Here is an example of the init method. 

    
  barney.Callbacky.init({
      verbose:true, // it logs all the work
      logger: window.console // set window.console as a logger
   });

  • trigger(key, arg)

    Call the function bound with the given string.

    On the trigger event, the corresponding function is called.

    Parameters
    ParamTypeDetails
    keystring

    Unique key to identify the function, used in bind() method before

    arg*

    Arguments to pass to saved function

    Example

    Trigger

    Here is an example of the bind/trigger method 

    // FIRST FILE
barney.Callbacky.bind('hello', function(args){ 
    console.log(args)) 
};

// SECOND FILE
barney.Callbacky.trigger('hello', 'world');
// console.log('world');
