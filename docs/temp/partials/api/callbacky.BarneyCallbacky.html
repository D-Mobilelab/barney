BarneyCallbacky
service in module callbacky

Description

Angular service of Callbacky module

Import & Usage

To use Callbacky service, you have to add BarneyCallbacky dependency to your component (directive, controller...). 

angular.module('mock').controller('HomePageController', [
    'BarneyCallbacky',
    function(BarneyCallbacky){
        // we can use "BarneyCallbacky" object here
    }
]);

Methods

  • bind()

    Refer to bind method of Analytics module, replacing barney.callbacky to BarneyCallbacky

  • clean()

    Refer to clean method of Analytics module, replacing barney.callbacky to BarneyCallbacky

  • init()

    Refer to init method of Analytics module, replacing barney.callbacky to BarneyCallbacky

  • trigger()

    Refer to trigger method of Analytics module, replacing barney.callbacky to BarneyCallbacky
