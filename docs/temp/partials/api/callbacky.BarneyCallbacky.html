BarneyCallbacky
service in module callbacky

Description

To use Callbacky service, you have to add BarneyCallbacky dependency to your component (i.e: directive, controller...).

In this example, I have added dependency of BarneyCallbacky to a controller: 

angular.module('mock').controller('HomePageController', [
    'BarneyCallbacky', '$scope',
    function(Callbacky, $scope){
        // we can use "Callbacky" object here
    }
]);
Note that I included BarneyCallbacky as dependency but I have renamed it as Callbacky to use it more easily in controller code.

List Methods:

  • bind
  • clean
  • init
  • trigger

Methods

  • bind(key, method)

    Bind one function to a string.

    By this way, on the trigger event (with the relative string), the bound function can be called everywhere in your code.

    Parameters
    ParamTypeDetails
    keystring

    Unique key to identify the function

    methodmethod

    Function to call when triggered

    Example

    Trigger

    Here is an example of the bind/trigger method 

    // FIRST FILE
Callbacky.bind('hello', function(args){ 
    console.log(args)) 
};

// SECOND FILE
Callbacky.trigger('hello', 'world');
// console.log('world');

  • clean(key)

    Remove the bound function from the given string

    Parameters
    ParamTypeDetails
    keystring

    Unique key to identify the function

    Example

    Clean

    Here is an example of the clean method 

    Callbacky.clean('hello');

  • init(options)

    This method is used to initialize the Callbacky module and it requires an object that contains two values

    Parameters
    ParamTypeDetails
    optionsObject

    (see attributes below)

    PropertyTypeDetails
    verboseboolean
    • true: the Callbacky module logs all the instruction done;
    • false: the Callbacky module logs nothing.

    (default: false)

    loggerObject

    Object that logs (i.e.: window.console, BarneyLogger, ...)

    (default: null)

    Example

    Here is an example of the init method. 

    
  Callbacky.init({
      verbose:true, // it logs all the work
      logger: window.console // set window.console as a logger
   });

  • trigger(key, arg)

    Call the function bound with the given string.

    On the trigger event, the corresponding function is called.

    Parameters
    ParamTypeDetails
    keystring

    Unique key to identify the function, used in bind() method before

    arg*

    Arguments to pass to saved function

    Example

    Trigger

    Here is an example of the bind/trigger method 

    // FIRST FILE
Callbacky.bind('hello', function(args){ 
    console.log(args)) 
};

// SECOND FILE
Callbacky.trigger('hello', 'world');
// console.log('world');
