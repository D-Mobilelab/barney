BarneyPromise
service in module promise

Description

Angular service of Promise module.

Import & Usage

To use Promise service, you have to add BarneyPromise dependency to your component (directive, controller...). 

angular.module('mock').controller('HomePageController', [
    'BarneyPromise',
    function(BarneyPromise){
        // we can use "BarneyPromise" object here
    }
]);

Methods

  • all()

    Refer to all method of Promise module, replacing barney.Promise to BarneyPromise.

  • any()

    Refer to any method of Promise module, replacing barney.Promise to BarneyPromise.

  • race()

    Refer to race method of Promise module, replacing barney.Promise to BarneyPromise.
