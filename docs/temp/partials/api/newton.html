newton

Description

Newton is the module for Newton.

You can use it to:

  • track a page
  • track an event
  • track the timespent on full website or page

Import & Usage

Vanilla JS

Import this single file to use Newton module, in Vanilla JS: 

<script type="text/javascript" src="./bower_components/barney/base/newton.js">
Now you can use global object barney.Newton and associated methods, described below.
You can start by initializing it with init method.

Angular

Import this single file to use Newton module, in Angular framework: 

<script type="text/javascript" src="./bower_components/barney/angular/newton.js">
Now you can import barney module for your Angular app. 
angular.module('yourApp', [ 'barney' ]);
Newton module for Angular contains only BarneyNewton service.

Methods

  • heartbeatsList()

    Get the list of all running heartbeats.

  • init(options)

    Init newton service.

    • enabled: enable or disable the module (default: true)
    • verbose: logs all the works done by the module (default: false)
    • logger: object that logs (i.e: window.console, BarneyLogger, ...) (default: null)
    Parameters
    ParamTypeDetails
    optionsObject

    list of options

    PropertyTypeDetails
    enabledboolean

    enable/disable tracking on Newton

    (default: true)

    verboseboolean

    enable/disable verbose mode, with logging

    (default: false)

    loggerObject

    logging methods to use for verbose mode

    (default: {})

  • isHeartbeatEnabled(keyword)

    Check if an heartbeat is running or not, returning true or false.

    Parameters
    ParamTypeDetails
    keywordstring

    Heartbeat name

  • startHeartbeat(keyword, options)

    Start a single Heartbeat.

    Parameters
    ParamTypeDetails
    keywordstring

    Heartbeat name

    optionsObject

    Heartbeat details

    Example

    Newton startHeartbeat

    Here is an example of the startHeartbeat method. 

    
  barney.Newton.startHeartbeat('aTestHeart', {
     label: 'page_title',
     valuable: 'no',
     action: 'no'
  });

  • stopAllHeartbeat()

    Stop all running heartbeats.

  • stopHeartbeat(keyword, options)

    Stop a single Heartbeat.

    Parameters
    ParamTypeDetails
    keywordstring

    Heartbeat name

    optionsObject

    Heartbeat details (can be also an empty object)

  • trackEvent(options)

    Track an event.

    Parameters
    ParamTypeDetails
    optionsObject

    Event details (i.e: ClickEvent, ShareEvent, ...)

    Example

    Newton trackEvent

    Here is an example of the trackEvent method. 

    
  barney.Newton.trackEvent('Click', {
     title: 'generated_event'
 });

  • trackPage(options)

    Track a page view event.

    Parameters
    ParamTypeDetails
    optionsObject

    Page details (i.e: title page, URL, ...)

    Example

    Newton trackPage

    Here is an example of the trackPage method. 

    
  barney.Newton.trackPage({
     title: 'page_pitle',
     URL: 'http://page_url'
 });
