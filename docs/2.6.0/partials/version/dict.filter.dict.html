dict
filter in module dict

Description

With BarneyDict you can print the value of a dictionary key but it hasn't to contain html code.

If dictionary key contains html code, use dict directive

Usage

In HTML Template Binding

{{ 'VALUED_KEY' | dict }}

In JavaScript

$filter('dict')()

Example

In this case I use it to print the value of a dictionary key in template (it hasn't to contain html code) 

  {{ 'VALUED_KEY' | dict }}
In this case I use it to print he value of a dictionary key as an attribute of an html tag (it hasn't to contain html code) 
   <input type="text" value="{{ 'VALUED_KEY' | dict }}" />
