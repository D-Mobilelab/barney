InfiniteScroll
directive in module infinite

Description

Allow infinite scroll on your page.

To use infinite scroll, you have to add infinite-scroll directive to the element that will contain the list of elements.

Furthermore, you have to pass a method to infinite-callback attribute: it will be executed when the bottom of the page will be reached.

If you want to reenable infinite scroll you have get first argument of this method and call it (see example below).

Furthermore, you can pass a boolean to infinite-enable to enable or disable infinite scroll; and an offset to infinite-offset to activate infinite scroll before the reaching of the end of the page.

Usage

as attribute
<ANY infinite-scroll>
   ...
</ANY>

Example

In this example, I have added infinite-scroll to a div that contains an ng-repeat: 

 // HTML
 <div infinite-scroll infinite-callback="functionToCall"  
  infinite-enable="flag" infinite-offset="100">
     <p ng-repeat="item in data">
          {{item}}
      </p>
 </div>

 // javascript
 $scope.flag = true;
