analytics

Description

Analytics is the module for Google Analytics.

You can use it to:

  • set user id
  • set a custom dimension
  • track a page
  • track an event

Import & Usage

Vanilla JS

Import this single file to use Analytics module, in Vanilla JS: 

<script type="text/javascript" src="./bower_components/barney/base/analytics.js">
Now you can use global object barney.analytics and associated methods, described below.
You can start by initializing it with init method.

Angular

Import this single file to use Analytics module, in Angular framework: 

<script type="text/javascript" src="./bower_components/barney/angular/analytics.js">
Now you can import barney module for your Angular app. 
angular.module('yourApp', [ 'barney' ]);
Analytics module for Angular contains only BarneyAnalytics service.

Methods

  • init(options)

    Init analytics service

    Parameters
    ParamTypeDetails
    optionsObject

    list of options

    PropertyTypeDetails
    enabledboolean

    enable/disable tracking on Google Analytics

    (default: true)

    verboseboolean

    enable/disable verbose mode, with logging

    (default: false)

    loggerObject

    logging methods to use for verbose mode

    (default: {})

    dimensionsObject

    list of custom dimensions that will be used in the app, where key is custom dimension name and value is slot id

    (default: {})

    Example

    Logger

    Logger with window.console 

    barney.analytics.init({
     verbose: true,
     logger: $window.console
});

    Logger with BarneyLogger 

    barney.logger.init({ enabled: true });
         
barney.analytics.init({
     verbose: true,
     logger: barney.logger
});
    Custom Dimensions

    In analytics initialization phase, you have to define all custom dimensions that you will use in application.

    Init method doesn't set custom dimension on Google Analytics, it only saves custom dimension for future use (for example in event tracking).

    You have to pass a pair (custom dimension name, slot id), where slot id is the slot of the custom dimension assigned from Google Analytics before.

    In this example, I set two custom dimensions ("UserStatus" with slot number 1 and "Valuable" with slot number 5): 

    barney.analytics.init({
    dimensions: {
        UserStatus : 1,
        Valuable : 5
    }
});

  • setDimension(dimensions)

    Set a user/session (not hit) custom dimension.

    Parameters
    ParamTypeDetails
    dimensionsObject

    where key is custom dimension name and value id custom dimension value

    Example

    The custom dimension has to be defined in init method before and, after, you have to use same custom dimension name. 

    // before, I save UserStatus and Valuable custom dimensions with slot "1" and "5"
barney.analytics.init({
    dimensions: {
        UserStatus: 1
    }
});
  
// after, I set UserStatus custom dimension with value "logged"
barney.analytics.setDimension({
    UserStatus : 'logged'
});

  • setId(id)

    Set analytics user id

    Parameters
    ParamTypeDetails
    idstring

    user id

  • trackEvent(options)

    Track an event

    Parameters
    ParamTypeDetails
    optionsObject

    (see attributes below)

    PropertyTypeDetails
    categorystring

    event category

    actionstring

    event action

    labelstring

    event label

    valueinteger

    event value

    dimensionsObject

    list of custom dimensions for this event tracking, where key is custom dimension name and value is custom dimension value

    Example

    barney.analytics.trackEvent({
    category: 'UI',
    action: 'open',
    label: 'menu',
    value: 7,
    dimensions: {
        Valuable: 'yes'
    }
});
    Note: the custom dimension (in this example Valuable) has to defined in init method before and you have to use same custom dimension name.

  • trackPage(options)

    To track a pageview

    Parameters
    ParamTypeDetails
    optionsobject

    (see attributes below)

    PropertyTypeDetails
    pagestring

    event page (e.g. '/category/7888')

    titlestring

    event page title (e.g. 'Home Page')

    dimensionsObject

    list of custom dimensions for this page tracking, where key is custom dimension name and value is custom dimension value

    Example

    barney.analytics.trackPage({
    page : '/category/7888',
    title: 'Category 7888 Page',
    dimensions: {
         Valuable: 'yes'
    }
     }
});
