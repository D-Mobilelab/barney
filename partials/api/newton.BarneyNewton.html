BarneyNewton
service in module newton

Description

To use Newton service, you have to add BarneyNewton dependency to your component (i.e: directive, controller...).

In this example, I have added dependency of BarneyNewton to a controller: 

angular.module('mock').controller('HomePageController', [
    'BarneyNewton', '$scope',
    function(Newton, $scope){
        // we can use "Newton" object here
    }
]);
Note that I included BarneyNewton as dependency but I have renamed it as Newton to use it more easily in controller code.

Methods

  • getSingleHeartbeat(keyword)

    This method is used to get a single heartbeat information, with its initial properties. You can use this method to retrieve all the information of an heartbeat.

    Parameters
    ParamTypeDetails
    keywordstring

    Newton heartbeat identifier

    Example

    Newton getSingleHeartbeat

    Here is an example of the getSingleHeartbeat method. 

    
      //get all the heartbeats information
      console.log('Heartbeats list: ', Newton.getSingleHeartbeat('atestheart'));

  • heartbeatsList()

    This method provides you the complete list of all the running heartbeat, with their initial properties. You can use this method to retrieve all the heartbeats information.

    Example

    Newton heartbeatsList

    Here is an example of the heartbeatsList method. 

          //get all the heartbeats information
      console.log('Heartbeats list: ', Newton.heartbeatsList());

  • init(options)

    This method is used to initialize the Newton module. It requires 3 parameters:

    • enabled: enable or disable the module (default: true)
    • verbose: logs all the works done by the module (default: false)
    • logger: object that logs (i.e: window.console, BarneyLogger, ...) (default: null)
    Parameters
    ParamTypeDetails
    optionsObject

    (see attributes below)

    PropertyTypeDetails
    enabledboolean
    • true: enable the module
    • false: disable the module

    (default: true)

    verboseboolean
    • true: log all the work
    • false: do nothing

    (default: true)

    loggerObject

    Object used to log (i.e: window.console, BarneyLogger, ...)

    (default: null)

    Example

    Logger

    Logger with window.console 

      Newton.init({
     enabled: true,
     verbose: true,
     logger: BarneyLogger
 });

    Logger with BarneyLogger 

    angular.module('mock').controller('HomePageController', [
    'BarneyNewton', 'BarneyLogger', '$scope',
    function(Newton, Logger, $scope){
  
        Logger.init({ enabled: true });
         
        Newton.init({
            enabled: true,
            verbose: true,
            logger: Logger
        });
  
    }
]);

  • startHeartbeat(keyword, params)

    This method is used to send the heartbeat starting event. Once called Newton will send message to track the time spent on the site with the interval specified in Newton. Is recommended to call the stopHeartbeat function when you want to stop heartbeat tracking.

    Parameters
    ParamTypeDetails
    keywordstring

    Newton heartbeat identifier

    paramsObject

    Newton custom_data to track what you want.

    Example

    Newton startHeartbeat

    Here is an example of the startHeartbeat method. 

    
  Newton.startHeartbeat('aTestHeart', {
     label: 'page_title',
     valuable: 'no',
     action: 'no'
  });

  • stopAllHeartbeat()

    This method is used to stop all the heartbeat that are running. You can use this method to easily stop all the heartbeat event.

    Is recommended to use it on the unload event of the window.

    Example

    Newton stopAllHeartbeat

    Here is an example of the stopAllHeartbeat method. 

    
   window.addEventListener("unload", function(){
      console.log('Document closed - stopping all heartbeat');
      Newton.stopAllHeartbeat();
   }, false);

  • stopHeartbeat(keyword)

    This method is used to send the heartbeat stop event. Once called Newton will stop to track the time spent on the site, and the heartbeat tracking will be removed from the heartbeats list. Before calling the stop heartbeat event remember that the heartbeat must be started.

    Parameters
    ParamTypeDetails
    keywordstring

    Newton heartbeat identifier

    Example

    Newton stopHeartbeat

    Here is an example of the stopHeartbeat method. 

    
  //after Newton.startHeartbeat(...)
  Newton.stopHeartbeat('aTestHeart');
